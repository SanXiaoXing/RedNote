# 内容管理仓库说明

## 📂 项目概述

本仓库用于存储由**抖音平台热搜话题**生成的小红书平台内容素材，包含以下两类资源：

- **小红书HTML文档**：基于当日抖音热点生成的格式化文案
- **关联图片资源**：与HTML内容配套的视觉素材

内容按自然日进行归档管理，便于后续检索与复用。

---

## 🗂 目录结构

```
.
├── README.md
├── 2025-2-10/                # 按日期建立独立文件夹
│   ├── 2025-2-10.html        # 小红书html内容
│   └── assets/                 # 当日图片素材
│       ├── cover.png
│       └── detail_shot.jpg
├── 2025-2-09/
│   ├── 2025-2-09.html
│   └── assets/
...
```
